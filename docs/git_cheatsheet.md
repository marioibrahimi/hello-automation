My Explanation: A new file exists, but Git is not tracking it yet

## unstaged
My Explanation: A tracked file changed, but is not prepared for commmit yet

## staged
My Explanation: File/change is prepared for the next commit

## working tree clean
My Explanation: no uncommitted changes in the repo
